# Скрипт для сбора ссылок

Скрипт собирает ссылки из поисковой выдачи гугла и яндекса, а так же со страниц сайтов поисковой выдачи

## Порядок использования

1. Установить зависимости
```py
pip3 install -r requirements.txt
```

2. Добавить в переменные окружения значение `LOG_LEVEL` - уровень логирования (NOTSET, DEBUG, INFO, WARNING, ERROR, CRITICAL)
Вместо прописывания переменной в окружение, можно положить в папку проекта файл `.env` в потором прописать
```dotenv
LOG_LEVEL=INFO  # NOTSET, DEBUG, INFO, WARNING, ERROR, CRITICAL
```

3. Запустить скрипт с нужными аргументами
```shell script
search_links.py выхухоль google -l 50 -r
```

Чтобы узнать значение и варианты аргументов, нужно набрать
```shell script
app.py -h
```

Если при запуске скрипта шелл выдает ошибку разрешений `permission denied`, нужно дать права на запуск файлу `app.py`
```shell script
chmod u+x search_links.py
```
